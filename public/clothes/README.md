# 服の画像フォルダ

このフォルダに服の画像をカテゴリ別に配置すると、アプリケーションのプリセットギャラリーから選択できるようになります。

## フォルダ構成

```
public/clothes/
├── tops/      # トップス（シャツ、ブラウス、セーターなど）
├── bottom/    # ボトム（パンツ、スカートなど）
└── set/       # セットアップ（上下セット、スーツなど）
```

## 推奨事項

- **画像形式**: PNG、JPG、JPEG、WebP
- **画像サイズ**: 10MB以下
- **背景**: 無地の背景で撮影された商品写真が最適
- **命名規則**: わかりやすいファイル名を使用
