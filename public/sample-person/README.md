# サンプル人物画像フォルダ

このフォルダに人物の全身写真を配置すると、アプリケーションのサンプルギャラリーから選択できるようになります。

## 画像の追加方法

### 簡単！ファイルをコピーするだけ

このフォルダに人物の全身写真をコピーするだけで、自動的にギャラリーに表示されます：

```bash
cp path/to/person.jpg public/sample-person/
```

**コードの編集は不要です！** ページをリロードすると、新しい画像が自動的に表示されます。

## 推奨事項

- **画像形式**: PNG、JPG、JPEG、WebP
- **画像サイズ**: 10MB以下
- **撮影**: 全身がはっきりと写っている写真
- **背景**: できるだけシンプルな背景
- **服装**: 上半身がはっきり見える服装
